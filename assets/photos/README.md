# 花材小柜子照片目录

把照片放在这里，后面网站正式接入文件数据时会从这些目录读取。

建议这样分类：

- `bouquets/`：整束花、花束作品、送花记录
- `flowers/`：已经确认名称的单个花材
- `unknowns/`：还没确认名称的花材

建议命名方式：

```text
2026-06-11-weekend-softness-01.jpg
2026-06-11-weekend-softness-02.jpg
eustoma-pink-01.jpg
unknown-purple-spike-01.jpg
```

先用英文、数字、短横线命名，避免空格和中文路径带来的兼容问题。
